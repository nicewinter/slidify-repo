Project Overview

  1. Use Shiny to realise interactive online data product
  2. This Shiny application is Iris Species Prediction based on user inputs
  3. Inputs: user can specify four features of Iris: Sepal.Length, Sepal.Width, Petal.Length, Petal.Width via web page input boxes (via ui.R)
  4. Outputs: the predicted Iris Species (setosa, versicolor, virginica) is shown (via ui.R), which is reactive to whatever the plant feature value inputs
  5. The machine learning algorithm from classification tree is used to process ui inputs at background to produce prediction results (via server.R)
  6. The iris R dataset package is employed for this data product for simplicity
  7. This pitch is done in Slidify

Conclusions

  1. Shiny is a powerful tool to quickly enable interactive data products online
  2. Users of Shiny need little or none knowledge of html and css to make web pages
  3. Two key parts of Shiny: GUI is implemented by ui.R whereas service logics are done by server.R at background. Most efforts focus on server.R
  4. People can take advantage of Shiny to present and publish their data products to a mass population, for knowledge sharing and reproducible research
